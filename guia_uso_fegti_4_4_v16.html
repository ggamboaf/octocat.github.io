Guía de Uso

Módulo de Gestión de Inventario Avanzado

Introducción

Esta guía proporciona instrucciones detalladas para la instalación, configuración y uso del Módulo de Facturación Electrónica GTI v4.4 para Costa Rica en Odoo. Este módulo permite la integración entre Odoo y los servicios de GTI Costa Rica, facilitando la automatización de procesos relacionados con facturación electrónica, validación de documentos tributarios y comunicación con la plataforma del Ministerio de Hacienda.

1. Instalación

  1. Descargue el módulo desde Odoo Apps.
  2. Extraiga el archivo ZIP y copie la carpeta del módulo en el directorio de addons de su instalación de Odoo.
  3. Reinicie el servidor de Odoo.
  4. Vaya a la aplicación **Apps** en su instancia de Odoo y busque el módulo por su nombre.
  5. Haga clic en **Instalar**.

2. Configuración

Para configurar las funcionalidades del módulo:

Ajustes

Vaya a Ajustes > GTI

  • Configuración de cuenta de GTI

    A continuación debes rellenar los campos obligatorios que obtienes del portal de GTI.
    • Número de Cuenta
    • Usuario
    • Clave
    • Código de actividad

  • Opción de envío manual de documentos

    Si habilita esta opción, el envío de todo documento a GTI pasa a hacer de forma manual mediante un botón que se visualiza en la parte superior del formulario de los documentos con el texto "Enviar a GTI", para habilitar este botón se debe seleccionar el check con el texto "Enviar a GTI" que se encuentra en el cuerpo del formulario de los documentos.

  • Configuración de APIs

    • Configuración API Hacienda
      • En este campo se ingresa la URL de API de hacienda que se utiliza en consultas para obtener el nombre, actividades económicas y poder agregar las exoneraciones del cliente.
      La URL por defecto es https://api.hacienda.go.cr
      También se utiliza para poder consultar el tipo de cambio.
    • Configuración API BCCR
      • Este servicio por defecto viene deshabilitado.
      La utilidad principal de este servicio es funcionar como segunda opción a la hora de actualizar los tipos de cambio.
      Si se habilita este servicio, pasa a ser el principal a la hora de actualizar el tipo de cambio.
      El beneficio de este servivio es que maneja mas tipos de cambio aparte del Dólar de los Estados Unidos de América.
      Para poder habilitar este serivio debes serguir los siguentes pasos:
      • La URL por defecto es https://apim.bccr.fi.cr/SDDE
      • Ingresar a esta página en la parte superior e ingresar donde dice Iniciar sesión.
      • Después ingresar donde dice Registrarse y completar el registro.
      • Al finalizar el registro y entrar al sistema con tu usuario, debes ingresar donde dice "Mi perfil" y en la parte de abajo en el apartado de Generar token debes darle al botón "Generar"
      • Al generar el token debe de aparecer una ventana de notificación donde va a estar un botón con el texto "Copiar token".
        Después de presionar dicho botón, debes pegar dicha información en el campo con el nombre "Token" en odoo

Módulo Facturación/Contabilidad

Vaya a Facturación/Contabilidad > Configuración

  • Facturación > Condiciones de pago

    Tome en cuenta esto si ya tiene "Condiciones de pago" creadas o si va a crear nuevas
    • Debe seleccionar un "Tipo de pago"
    • Si selecciona el tipo de pago es "Crédito", debe asignar un plazo en días para el pago

  • Contabilidad > Impuestos

    Tome en cuenta esto si ya tiene "Impuestos" creados o si va a crear nuevos
    • Debe seleccionar un "Código de impuesto"
    • Debe seleccionar un "Código de la tarifa del impuesto"

  • GTI > Actividades económicas

    Las actividades económicas se crean solas cuando se crea al cliente, pero si el servicio de hacienda está caído, se ocupa crear las actividades económicas manualmente.
    • Debe agregar el código y la descripción de la actividad

  • GTI > Unidades de medida

    Aquí se crean y editan las unidades de medida
    • Debe agregar el código, la descripción y el símbolo de la unidad de medida
    • En el siguiente enlace podrá encontrar la documentación sobre las unidades de medida que se encuentran en el documento llamado ANEXOS Y ESTRUCTURAS_V4.4.pdf

  • GTI > Cabys

    Aqui se crean y editan los códigos Cabys
    • Debe agregar el código, la descripción del bien o servicio y el símbolo del cabys
    • En el siguiente enlace podra encontrar la documentacion las unidades de medida se encuentran en el documento llamado ANEXOS Y ESTRUCTURAS_V4.4.pdf

  • GTI > Medios de pago

    Aqui se crean y editan los medios de pago
    • Debe agregar el código, la Descripción del medio de pago
    • En el siguiente enlace podra encontrar la documentacion los medios de pago

  • GTI > Provincias

    Aqui se crean y editan las provincias
    • Debe agregar el código, el nombre de la provincia
    • En el siguiente enlace podra encontrar la documentacion las provincias

  • GTI > Cantónes

    Aqui se crean y editan los cantónes
    • Debe agregar el código, el nombre del cantón y la provincia
    • En el siguiente enlace podra encontrar la documentacion los cantónes

  • GTI > Distritos

    Aqui se crean y editan los distritos
    • Debe agregar el código, el nombre del distrito y el cantón
    • En el siguiente enlace podra encontrar la documentacion los distritos

Módulo Facturación/Contabilidad

Vaya a Facturación/Contabilidad > GTI

  • Sucursal > Sucursales

    Aquí se crean y editan las sucursales
    • Debe agregar el nombre de la sucursal
    • Debe agregar la provincia, el cantón, el distrito
    • Debe agregar el identificador de sucursal que se optiene desde GTI
    • Debe agregar el teléfono y el Correo
    • El apartado de "Terminales" tambien se puede entender como cajas que se crean desde GTI
      • Debe agregar el identificador de terminal
      • Debe agregar un nombre
      • Debe agregar el ususario de odoo asociado a esa terminar

3. Uso Básico

Módulo Facturación/Contabilidad

Vaya a Facturación/Contabilidad > Clientes

  • Clientes

    Aquí se crean y editan los clientes
    • Al darle en el botón con el texto "Nuevo", se abre una ventana para poder crear al cliente
      • Se debe agregar la identificación
      • Se debe agregar el teléfono
      • Se debe agregar el correo

    • Después de agregar la anterior informacion se debe dar al botón con el texto "Buscar", y esto rellena el resto de información consultando a Hacienda
    • Después de darle al botón con el texto "Buscar", se debe dar al botón con el texto "Crear" para poder agregar al cliente.
    • Esto te llevará al formulario del cliente recién creado
    • En el formulario podra modificar datos pertinentes del cliente

  • Productos

    Aquí se crean y editan los productos
    • Al darle en el botón con el texto "Nuevo", se abre una ventana para poder crear el producto
      • Se debe agregar el nombre
      • Se debe agregar el cabys
      • Se debe agregar la unidad de medida
