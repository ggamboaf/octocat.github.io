Módulo de Integración Odoo - Pasarela de Pago Externa

Descripción General

Este conector facilita la integración fluida entre Odoo y una pasarela de pago externa, permitiendo que los pagos de las ventas realizadas en la web o a través de pedidos de Odoo se sincronicen de manera automática y segura.

Características Principales

  • Sincronización automática de pedidos y estados de pago.
  • Manejo de transacciones seguras y encriptadas.
  • Configuración flexible para diferentes pasarelas.
  • Registro de historial de transacciones en Odoo.
  • Compatibilidad con Odoo versión 16.
