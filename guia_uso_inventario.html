Guía de Uso

Módulo de Gestión de Inventario Avanzado

Introducción

Esta guía proporciona instrucciones detalladas para la instalación, configuración y uso del Módulo de Gestión de Inventario Avanzado en Odoo. El módulo está diseñado para optimizar la trazabilidad de productos mediante la gestión de lotes y números de serie.

1. Instalación

  1. Descargue el módulo desde Odoo Apps.
  2. Extraiga el archivo ZIP y copie la carpeta del módulo en el directorio de addons de su instalación de Odoo.
  3. Reinicie el servidor de Odoo.
  4. Vaya a la aplicación **Apps** en su instancia de Odoo y busque el módulo por su nombre.
  5. Haga clic en **Instalar**.

2. Configuración

Para configurar las funcionalidades del módulo:

  • **Habilitar Lotes y Números de Serie:** Vaya a **Inventario > Configuración > Ajustes** y active las opciones "Gestión de Lotes" y "Gestión de Números de Serie".
  • **Configuración de Productos:** En la ficha de cada producto, configure si utiliza lotes o números de serie.

3. Uso Básico

Durante las operaciones de recepción y entrega de productos, el sistema le pedirá que ingrese los lotes o números de serie correspondientes.

  • **Recepción de Productos:** Al validar una recepción, aparecerá una ventana emergente para registrar los lotes/números de serie.
  • **Informes de Trazabilidad:** Use el menú de informes para generar reportes que le permitan rastrear el origen y destino de cada lote o número de serie.
